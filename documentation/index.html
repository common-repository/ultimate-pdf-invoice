A) Introduction - top

Ultimate PDF Invoice Plugin is an easy, quick and automatically generate and attach customizable PDF Invoices to WooCommerce emails.

Ultimate PDF Invoice Plugin gives you ability to automatically generate pdf with live editor. Also you can manage pdf invoice templates and contents

Ultimate PDF Invoice Plugin plugins inbult functionality of generate any order pdf by order page or plugin page.

B) Installation - top

Ultimate PDF Invoice requires WordPress 3.7+

Ultimate PDF Invoice Plugin has been successfully tested with WooCommerce 2.1.x, 2.2.x, 2.3.x, 2.4.x., 2.5.x.

Just like every plugins, just copy the ultimate-pdf-invoice folder in your wp-content/plugins folder then activate it through WordPress dashboard. The plugin is only accessible for Admin's

If the installation went OK, you can now access the plugin Dashboard from the 'Ultimate Invoice' menu.

You should see four sub menu in page. Just click it for what you want.

D) Live Editor - top

This Plugin allow you to perform live editing.change value any field and you are done.

For save editor value just click save button and check preview of your changes..

E) Template Preview - top

This Plugin allow you to see export data preview before you activate template. Just click on 'Preview' button thats it, you can see what happen..

F) Settings - top

This Plugin allow you to activation of templates and other settings.

Settings options

a) Template activation

You can choose which template is activate for send e-mail.

b) Email Attachment

Invoice is atached with email or not. if set to do not attach than it will not attach any invoice at any order place

c) Attach Invoice

Invoice attach with which event. select any event from list and you are done!!!

G) Download Pdf - top

This Plugin allow you download any pdf invoice, just select any order id and click on download button and you are done!!!.
